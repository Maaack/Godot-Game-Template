# Updating Credits

These instructions cover updating the credits / attribution.

Credits are parsed automatically from the `ATTRIBUTION.md` file included with the template.

## Basics

1.  Update the example `ATTRIBUTION.md` with the project's credits.
2.  Open `credits_label.tscn`.
3.  Check the `CreditsLabel` has updated with the text.
    1.  Optionally, disable `Auto Update` and customize the text.
4.  Save the scene (even if it shows no changes).

Optionally, `*.md` can be included for export in the _Export_ window's `Resources` tab, and the steps to update `credits_label.tscn` can be skipped.
